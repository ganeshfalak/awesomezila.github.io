CSS Selectors Reference


Selectors


Example Name Description
p {
   color: green;
} 		Type Selector The type selector selects <p> element and styles according to the declarations.
* {
   font-family: Arial;
} 		Universal Selector The universal selector selects all elements of any type.
.brand {
   
} 		Class Selector The class selector selects the element which has class attribute specified with a name.
#main-heading {
   
} 		ID Selector The id selector selects the specific HTML element which has id attribute with a unique name in order to provide uniques styles to it.
[href] {
   color: orange;
} 		Attribute Selector The attribute selector selects the element which contains specific attribute in it.
p:hover {
   background-color: lime;
} 		Pseudo-class A Pseudo-class can be attached to any selector and states of it changes upon user interaction/behaviour accordingly.
